# 课程设计评分标准

## 课程设计功能完成情况评分标准（满分8）
①	独立完成，主题明确，页面上所有功能均与主题有关，页面样式简洁大方、美观，用户操作体验好。构建完整的项目，文件命名规范，项目目录结构合理。（扣分项）
②	体现各种指令的使用、事件响应、计算属性等。（2分）
③	定义了合适的组件并使用了组件注册等功能。（2分）
④	定义恰当的路由（定义、嵌套、守卫等），实现页面的跳转。（2分）
⑤	使用Pinia管理数据，以及使用axios完成http请求。（2分）

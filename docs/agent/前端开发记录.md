## 已确定的关键决策

- **snake_case** 全链路（Mock = DB = 后端 JSON）
- **暗色扁平**主题（`#0a0a0a`，`var(--color-accent)` 引用 tokens）
- 点击曲目/试听 → `playAlbumTracks`（**替换**播放列表），不是追加
- `addAlbumTracks` 用于从首页"试听"按钮（追加到末尾）
- 播放器 `position: fixed; bottom: 0`，主内容 `padding-bottom: 80px`
- 专辑卡片网格宽度问题记录在 `CONTEXT.md` 已知问题

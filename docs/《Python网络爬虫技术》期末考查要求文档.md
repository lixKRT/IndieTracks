三、项目功能技术要求（全覆盖考核评分点）
学生作品必须完整覆盖课程六大核心考核模块，所有功能需自主实现、可正常稳定运行，具体技术要求如下：
1. 爬虫框架项目搭建与配置（0-20分）
项目需使用Python主流爬虫框架开发（Scrapy优先，也可使用PySpider等正规爬虫框架），禁止仅使用原生requests简单拼接代码应付考核。要求完成完整项目初始化搭建，规范项目目录结构，同时完成基础工程配置，包含但不限于：请求头UA伪装、访问延时配置、基础反爬适配、请求异常捕获、失败重试机制、日志输出配置等。代码结构规范、注释完整清晰，项目可直接部署运行，依赖环境完整可复现。
2. 网页基础数据爬取（0-23分）
基于HTTP/HTTPS协议对目标网站发起合法网络请求，能够稳定获取网页响应内容。爬取过程无频繁报错、无主动中断问题，可完整抓取目标页面核心原始数据。能够应对网站基础反爬策略，规避基础拦截、访问限制问题，保证单次页面爬取数据完整、内容不缺失，爬取结果稳定可靠。
3. JavaScript动态渲染数据获取（0-10分）
所选爬取目标必须包含JS动态渲染加载的数据（Ajax异步加载、前端渲染数据等），禁止选择纯静态HTML页面。需通过Selenium、Playwright或接口逆向解析等合规技术，正确抓取页面动态加载内容，实现动态数据完整采集，解决静态请求无法获取动态内容的问题，保证动态字段数据完整、准确。
4. 多页数据批量爬取（0-7分）
项目必须实现自动化分页爬取功能，通过解析分页参数、URL规则、翻页触发逻辑等方式，完成至少10页及以上有效页面的自动化批量爬取。翻页逻辑稳定，无重复爬取、漏页、提前终止等问题，可批量采集多页结构化数据，实现批量数据采集自动化。
5. 数据解析与精准提取（0-20分）
使用XPath、BeautifulSoup、正则表达式、PyQuery等课程所学解析技术，对爬取的原始网页数据进行清洗、过滤、解析。需精准提取至少5项有效结构化字段（如标题、发布时间、作者、链接、内容、价格、分类等），剔除空白数据、冗余代码、无效字符，完成数据清洗规整，最终输出规范、整洁、无错误的结构化数据。
6. 数据库数据持久化存储（0-20分）
禁止仅使用txt、csv等本地文件存储，必须使用数据库完成数据持久化存储。可选用MySQL、SQLite、MongoDB等主流数据库，自主设计数据表/集合结构，合理设置字段类型。实现数据批量写入、数据去重、存储异常处理等功能，保证爬取的结构化数据完整、有序、无缺失、无重复，数据库可正常查询、读取已存储数据。
